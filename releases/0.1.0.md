# Release v0.1.0 — Initial release

**Date:** 2026-02-01

## Summary
Initial release of the `testkey` project. This release includes the basic project structure, environment variable loader, and tests.

## Notable changes
- Added `testkey.py` with `load_env()` to read `KIMI`, `DSEEK`, and `OPENAI` environment variables.
- Added pytest-based unit tests in `tests/test_testkey.py`.
- Added `pyproject.toml`, `dev-requirements.txt`, and `requirements.txt`.
- Added GitHub Actions CI workflow (`.github/workflows/ci.yml`) to run tests on push and PR.
- Created initial `README.md` with Quickstart instructions.

## Verification
- All tests pass locally and in CI configuration (two tests passing with pytest).

## Notes
This is the first release; future releases will include additional features, improved test coverage, and linting/formatting tools.
